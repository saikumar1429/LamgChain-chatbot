# LangChain Chatbot using Hugging Face

This project is a Streamlit-based conversational chatbot built using LangChain and Hugging Face Inference Endpoints. It allows users to interact with a large language model by entering their Hugging Face API token at runtime.

The chatbot maintains conversation history using Streamlit session state and generates responses using the Mistral-7B-Instruct model.

---

## Project Overview

This application demonstrates how to:

- Build a conversational AI chatbot
- Integrate Hugging Face models with LangChain
- Use runtime API key authentication
- Maintain chat history in a Streamlit app
- Create prompt templates using LangChain

The system uses the Hugging Face hosted model:
mistralai/Mistral-7B-Instruct-v0.2

---

## Features

- Runtime Hugging Face API key input
- Interactive chat interface using Streamlit
- Conversation memory using session state
- Prompt templating with system and user roles
- Output parsing using LangChain
- Adjustable temperature and token limits
